# 画面一覧

## MVPで作る画面

| 画面 | 目的 | 優先度 |
| --- | --- | --- |
| トップ画面 | アプリ概要とルーム作成導線を見せる | 中 |
| ルーム作成画面 | ホストがイベント用ルームを作る | 高 |
| 参加画面 | 参加者が名前だけで参加する | 高 |
| ルーム画面 | 注文入力・注文一覧・集計を確認する | 高 |
| ホスト管理画面 | 採用・却下・購入チェックを行う | 高 |
| 買い物リスト画面 | 採用済みアイテムだけを見る | 中 |

## 最初に作る順番

1. ルーム作成画面
2. 参加画面
3. 注文入力フォーム
4. 注文一覧
5. 集計表示
6. ホスト管理画面
7. 買い物リスト画面

## 画面ごとの最低条件

### ルーム作成画面

- イベント名を入力できる
- 日付を入力できる
- メモを入力できる
- 作成後に参加URLとホストURLを確認できる

### 参加画面

- 名前を入力できる
- ログインなしで参加できる
- 参加後にルーム画面へ進める

### ルーム画面

- 商品名・価格・数量・メモを入力できる
- 注文一覧を見られる
- 合計金額を見られる

### ホスト管理画面

- 提案を採用できる
- 提案を却下できる
- 採用済みアイテムに購入済みチェックを付けられる
