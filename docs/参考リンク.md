- 買い物リストアプリ: 買う物のチェックに強いが、誰が希望したかや金額集計は弱くなりやすい。

オーダールームでは、イベント前から当日までの「提案、採用、買い物リスト、集計」を1つのルームで扱う点を差分にする。
